# Online Examination System

This project is developed as part of the Oasis Infobyte Java Internship.

## Technologies Used
- Java
- JDBC
- MySQL
- VS Code

## Features
- User Login
- Update Profile and Password
- MCQ Based Quiz
- Timer and Auto Submit
- Logout Functionality

## Project Structure
src/ - Java source files  
lib/ - MySQL connector jar  
database/ - SQL file for database setup  

## How to Run
1. Import the database using database/mcq_system.sql
2. Add MySQL connector jar in lib folder
3. Compile the Java files
4. Run Main.java
