# Guess The Number Game (Java)

## Description

A simple Java game where the computer generates a random number between 1 and 100, and the user tries to guess it. The program gives hints if the guess is too high or too low until the correct number is guessed or attempts are finished.

## Features

* Random number generation (1–100)
* Limited number of attempts
* Multiple rounds
* Score calculation based on attempts
* Option to stop the game after any round

## Technologies Used

* Java
* JOptionPane (Swing)
* VS Code

## How to Run

1. Open the project in VS Code.
2. Run the file **GuessTheNumberGame.java**.
3. Follow the dialog box instructions to guess the number.
