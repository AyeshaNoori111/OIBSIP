# Online Reservation System

## Project Description
This is a console-based Online Reservation System developed using Java, JDBC, and MySQL.  
The system allows users to login, reserve train tickets, view reservation details, and cancel reservations.

## Technologies Used
Java  
JDBC  
MySQL  
VS Code  
GitHub  

## Features
- User Login
- Ticket Reservation
- View Reservation Details
- Cancel Reservation
- Database Integration using MySQL

## Database
Database Name: reservation_db

Table:
reservations

## How to Run

Compile:
javac -cp ".;lib/mysql-connector-j-9.6.0.jar" src/OnlineReservationSystem.java

Run:
java -cp ".;lib/mysql-connector-j-9.6.0.jar;src" OnlineReservationSystem
